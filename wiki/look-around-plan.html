LookAround Enhancement Plan

Done

Kitty Barrage · May 2026

Use Cases to Support

Curious kitty — Looks left and right quickly, multiple times. Short pauses. High energy.
speed=1.8 · reps=3 · pause_left=0.1 · pause_right=0.1
Cautious kitty — Looks left slowly, holds there a long time, then continues on its way without looking right at all.
speed=0.6 · reps=1 · left_only=true · pause_left=2.0
Lazy glance — One slow look both ways, long pause facing forward at the end before returning to the brain loop.
speed=0.7 · reps=1 · pause_center=1.5
Paranoid kitty (loop) — Loops continuously looking both ways fast until interrupted by movement.
speed=2.0 · loop=true · pause_left=0 · pause_right=0

Current Phase Sequence

Base sequence: center → right → right → left → left → center

With pauses and direction flags, the full proposed sequence:

center → right → pause right → right → left (opt) → pause left (opt) → left → center (opt) → pause center → repeat / exit

Optional phases are skipped if left_only, right_only, or pause = 0. Pause phases run in tick(delta) — no animation, just a timer counting down before moving to the next phase.

Design Questions

1
Direction preference

How should "left only" or "right only" be configured?

2
Repetitions — fixed or random range?

How many times does the full look-around cycle repeat before exiting?

3
Speed control — where does it live?

Playback speed affects how fast the animation frames advance. Godot's AnimatedSprite2D.speed_scale handles this. Where should it be set?

4
Where do all these parameters live?

There are now ~8 look-around parameters. Where should they be defined?

Debug Overlay Additions

New controls near Look Around button

  • Speed — HSlider 0.2–3.0, step 0.1, default 1.0
  • Pause Left — HSlider 0–3.0s, step 0.1
  • Pause Right — HSlider 0–3.0s, step 0.1
  • Pause Center — HSlider 0–3.0s, step 0.1
  • Reps — SpinBox 1–10
  • Look Left — CheckBox
  • Look Right — CheckBox

Layout idea

The button column would expand for Look Around with a collapsible sub-panel, or the controls appear inline below the button row. Sliders show current value as a label.

All values read at button-press time — not live-applied. So you can set params, then press Look Around to test that exact configuration.

Task List

Phase 1 · AnimationController speed support

  1. Add speed parameter to play_once and play_loop — sets _player.speed_scale before playing. _cancel_silent restores it to 1.0. scripts/AnimationController.gd

Phase 2 · KittyPersonality + LookAroundState

  1. Add Look Around export group to KittyPersonality — all 7 fields listed above. scripts/KittyPersonality.gd
  2. Rewrite LookAroundState with full phase sequence — 7-phase sequence, skips phases when pause=0 or direction disabled. Reads all params from entity.personality. scripts/kitty_states/LookAroundState.gd

Phase 3 · Debug overlay UI

  1. Add Look Around controls to debug overlay — speed slider, pause sliders × 3, reps spinner, look-left/look-right checkboxes. Values passed to LookAroundState at press time. scripts/DebugOverlay.gd
